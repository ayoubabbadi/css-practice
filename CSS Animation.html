CSS Animation, Transition & Transform Examples

1. Transition Examples

Transitions allow property changes to occur smoothly over a specified duration

Hover to see transition

Transition Properties:

  • transition-property: Specifies which properties to transition (or 'all')
  • transition-duration: How long the transition lasts (e.g., 0.5s)
  • transition-timing-function: Speed curve (ease, linear, ease-in, etc.)
  • transition-delay: Delay before transition starts

Transform Functions:

  • translate(x, y): Moves element along X/Y axes
  • rotate(deg): Rotates element by specified degrees
  • scale(x, y): Resizes element (1 = original size)
  • skew(x, y): Skews element along X/Y axes
  • transform-origin: Changes the origin point for transforms

3. Animation Examples

Animations allow complex multi-stage visual changes using @keyframes

Animation Properties:

  • @keyframes: Defines the animation sequence
  • animation-name: Name of the @keyframes rule
  • animation-duration: How long one cycle lasts
  • animation-timing-function: Speed curve (ease, linear, etc.)
  • animation-delay: Delay before animation starts
  • animation-iteration-count: How many times to play (or 'infinite')
  • animation-direction: normal, reverse, alternate, alternate-reverse
  • animation-fill-mode: Styles applied before/after animation
  • animation-play-state: running or paused
