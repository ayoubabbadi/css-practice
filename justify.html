Difference Between Justify Properties

1. justify-content

The justify-content property is used to align items along the main axis in a Flexbox or Grid container. It controls the distribution of space between and around items.

Example: justify-content in Flexbox

Item 1
Item 2
Item 3
.flex-container {
    display: flex;
    justify-content: space-between; /* Distributes items with equal space between them */
}

2. justify-content

The justify-content property is used to align items along the main axis in a Flexbox or Grid container. It controls the distribution of space between and around items.

Example: justify-content in Grid

Item 2
Item 3
Item 4
Item 5
Item 6
.grid-container {
    display: grid;
    grid-template-columns: repeat(3, 100px);
    justify-content: space-between; /* Distributes items with equal space between them 
}

3. justify-items

The justify-items property is used in CSS Grid to align items within their grid cells along the main axis.

Example: justify-items in Grid

Item 1
Item 2
Item 3
Item 4
Item 5
Item 6
.grid-container {
    display: grid;
    grid-template-columns: repeat(3, 100px);
    grid-template-rows: repeat(2, 50px);
    justify-items: center; /* Centers items within their grid cells */
}

4. justify-self

The justify-self property is used in CSS Grid to align a specific item within its grid cell along the main axis. It overrides the justify-items value for that item.
